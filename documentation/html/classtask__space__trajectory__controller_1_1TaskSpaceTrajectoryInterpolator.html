Interpolate between two points in the trajectory.

The interpolation method is specified by the interpolation_method_ member variable. If the interpolation method is NONE, the end point is returned. If the interpolation method is LINEAR, a linear interpolation of position, orientation and velocity is performed. The acceleration is ignored. The interpolation method used for orientation is SLERP.

Parameters
[in]start_pointThe start point of the interpolation.
[in]end_pointThe end point of the interpolation.
[in]time_from_startThe time from the start of the trajectory at which to interpolate between the two points.
[out]sampled_pointThe interpolated point.
[in]loggerThe logger for logging messages.
